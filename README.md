# TGtoolspanel
⚙️ Управляйте вашим ПК с помощью телеграм в любом месте!

# Функционал
1. Быстрый скриншот
2. Управление питанием ПК
3. Отложенное выключение ПК
4. Отправка\загрузка файлов
5. Управление процессами (Запуск, остановка)
6. Управление мышью
7. Отправка комбинаций нажатий клавиш
8. Отправка уведомлений на ПК
9. Отправка текста в буфер обмена

Управлять ботом имеет право только пользователь указанный в ADMIN_ID, у других пользователей при попытке использовать ваш бот будет показываться ошибка доступа.
ADMIN_ID заполняется автоматически, вводится ID пользователя который первый ввел /start в вашем боте. При необходимости ADMIN_ID можно заменить, 23 строка в bot.py

# Установка и запуск
1. Создание бота в @BotFather -
2. Вводим /newbot
3. Отправляем имя бота, Например - TG TOOLS PANEL
4. Отправляем логин бота, заканчивающийся на bot, Например - tgtoolspanel_bot
5. Вы получаете сообщение об успешном создании бота и также его API такого вида - 1234567:ASDFGHJKLQWERTY, копируем его.
6. Переходим в нашего бота и нажимаем запустить
7. Скачиваем с репозитория файлы bot.py и install.txt
8. Скачиваем и устанавливаем Python, тестилось на версии 3.9.13 (https://www.python.org/downloads/)
9. !!! ПРИ УСТАНОВКЕ ОБЯЗАТЕЛЬНО СТАВИМ ГАЛОЧКУ ADD PYTHON TO PATH БЕЗ ЭТОГО РАБОТАТЬ НЕ БУДЕТ !!!
10. Открываем bot.py через любой текстовый редактор (Использовал notepad++)
11. В самом вверху в настройках бота в кавычки вводим API своего бота в строку BOT_TOKEN (Вместо ВАШ_TELEGRAM_BOT_TOKEN_ЗДЕСЬ)
12. ADMIN_ID не трогаем, заполняется автоматически первого пользователя который введет /start в вашем боте
13. Сохраняем файл, закрываем.
14. Запускаем CMD (WIN+R и вводим cmd)
15. Пишем pip install -r и путь до файла install.txt который скачали (Например pip install -r C:\download\install.txt)
16. Ждем окончания загрузки всех библиотек.
17. Вводим cd и путь где лежит файл bot.py (Пример cd C:\download)
18. Вводим pyinstaller -w -F bot.py дожидаемся окончания
19. В той же папке где лежал bot.py создалась папка dist, копируем от туда bot.exe в любую удобную папку, я кидаю просто на диск C.
20. Жмем ПКМ по bot.exe - Отправить - Рабочий стол (создать ярлык)
21. Ярлык закидываем в автозапуск (WIN+R вводим shell:startup и в эту папку кидаем наш ярлык)
22. Перезагружаем ПК или вручную запускаем этот ярлык от имени админа для первого запуска бота.
23. В дальнейшем бот будет автоматически запускаться с запуском Windows.
24. Наслаждаемся.
